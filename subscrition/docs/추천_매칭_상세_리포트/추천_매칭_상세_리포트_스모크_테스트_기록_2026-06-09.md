## LH 수집 및 AI 파싱 결과

### Candidate 수집

- `GET /api/admin/import/lh/candidates/collect?page=1&size=1`
- 결과: `200`
- 토지 공고 skip 로직 정상 확인

### Selected force import

- `POST /api/admin/import/lh/selected`
- request body: `{"candidateIds":[1],"force":true}`
- 최종 결과:

```json
{
  "fetched": 1,
  "scanned": 1,
  "skippedLand": 0,
  "skippedCommercial": 0,
  "unchanged": 0,
  "geminiSkipped": 0,
  "imported": 1,
  "reparsed": 1,
  "failed": 0
}
```

### AI parsing 특이 사항

- 1차 재시도 중 Gemini가 일시적으로 `503 high demand`를 반환했다.
- 재시도 후 Gemini PDF parse 경로가 정상 진행됐다.
- 최종 DB fingerprint 상태는 `PARSED`이다.

## 후속 Hardening Smoke

### selected LH import candidateIds 상한

- `POST /api/admin/import/lh/selected`
- 조건: `candidateIds` 101개
- 결과: `400`

```json
{

}
```

### public-visible status 공통화

- public-visible 공고 검수 상태는 `ParseReviewStatus.publicVisibleStatuses()`로 공통화했다.
- 적용 status는 기존과 동일하게 `APPROVED`, `CORRECTED`이다.
