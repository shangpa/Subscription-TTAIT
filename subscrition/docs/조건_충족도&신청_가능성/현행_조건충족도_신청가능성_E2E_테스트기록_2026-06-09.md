## 11. 특이사항 및 후속 고려

| 항목 | 내용 | 영향 |
| - | - | - |
| 테스트 데이터 잔존 | MySQL volume에 smoke 데이터가 남아 있다. | 필요 시 별도 정리 가능 |
| MySQL 종료 | 테스트 완료 후 `docker compose stop mysql`로 컨테이너를 중지했다. | volume 데이터는 유지 |
| 신청 기간 | 테스트일 기준 접수 전이라 `APPLICATION_PERIOD=NEEDS_VERIFICATION`이 정상이다. | 기능 정상 범위 |
| geocoding | public detail unit 일부 `geocodeStatus=FAILED`가 관찰됐다. | checklist 기능에는 영향 없음 |
| cache header | checklist는 사용자별 GET 응답이다. | `Cache-Control: private, no-store` 적용 완료 |
