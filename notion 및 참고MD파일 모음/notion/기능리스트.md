# 기능리스트

프로젝트 이름 : 청약 따잇

주제 : 청년·신혼부부·무주택자·고령자·저소득자 등 **공공임대 수혜 대상자**를 위한 맞춤형 공고 알림 서비스.

핵심 차별점 : 공고를 보여주는 데서 그치지 않고, **"이 공고, 나 신청 가능해? 붙을 수 있어?"** 까지 AI가 대신 판단해주는 서비스.

- 개발 환경
    - 백엔드 : Java – SpringBoot
    - 프론트 : JS – React , Vue (구현 방식 미정)
    - DB : MySQL + Redis

관리자 기능

1. 공고 데이터 수집 및 저장
    - api 공고 저장
        1. Spring Boot에서 외부 api 받아옴 
        2. 받아온 api의 json파일에서 pdf 다운로드 링크 확인 
        3. 다운로드 링크에서 Spring Boot가 PDF 다운로드
        4. ai API 등 PDF 전달 및 JSON 응답 요청 
        5. 응답받은 JSON을 DB에 저장
    - api에서 못받아온 정보를 가져올때
        - 웹크롤링 (예 : 평면도)
    - PDF 는 어떻게 할껀가?
        - api에서 받아온 pdf를 ai사용해서 어느정도로 읽어오는지 확인하고
        - api에서 받아온 json이랑 pdf에서 ai가 읽어온 받아온 json을 합쳐서 db에 어떻게 저장할 것인가?
        - 이걸 ai를 써서 파싱할껀지 아니면 로직으로 파싱할껀지

사용자 기능

1. 회원가입 및 프로필 관리 
    - 회원가입 : 아이디, 비밀번호, 전화번호, 이메일
    - 프로필
        - 신분 유형 선택 (청년 / 신혼부부 / 무주택자 / 고령자 / 저소득자 / 다자녀)
            - 예시
                
                
                | 유형 | 대상 |
                | --- | --- |
                | 🧑 청년 | 만 19~39세, 미혼 |
                | 💑 신혼부부 | 혼인 7년 이내 |
                | 🏠 무주택자 | 무주택세대구성원 |
                | 👴 고령자 | 만 65세 이상 |
                | 💰 저소득자 | 기초수급자, 차상위 |
                | 👨‍👩‍👧‍👦 다자녀 | 미성년 자녀 3인 이상 |
            1. 청년/ 신혼부부/ 고령자 / 다자녀 에서 선택
            2. 무주택자 선택
            3. 저소득자
                - 소득 정보: 가구원수(1인~7인 이상), 세전 월평균 소득액 적기
                - 취약계층 여부: 수급자, 차상위계층, 한부모가족 여부 체크박스
                - 자산 정보: 총 보유 자산 및 차량 보유 가액 (AI 판단의 핵심 변수)
        - 세부 정보 입력 (나이, 혼인여부, 자녀수, 소득, 자산, 희망지역, 희망주택유형)
2. 전체 공고 리스트업 및 검색
    - 지역, 가격, 신분 유형(프로필), 공고 이름으로 검색, 시공사(SH,LH등)
3. 개인화 공고 알림 (카톡 알리미)
    - 저장된 공고 데이터를 바탕으로 사용자에서 알맞은 공고를 카톡 알리미로 보낸다
    - 알리미로 간 공고를 즐겨찾기 해두면 마감 7, 3, 1일전에 마감 알림을 보낸다.
4. 맞춤형 공고 추천 리스트
    - 저장된 공고에서 사용자에게 맞는 공고를 리스트업한다
    - 추가기능 : 공고 지도에서 알려주기
        - 사용자에게 추천한 공고를 지도에서 마커로 표시해주는 기능
5. AI 당첨 가능성 리포트
    - 사용자가 선택한 공고의 당첨 가능성을 분석해서 전달
